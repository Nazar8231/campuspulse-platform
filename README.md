# CampusPulse

CampusPulse — локальна вебпрограма для моніторингу самопочуття студентів, проходження психологічних тестів, перегляду результатів і роботи з аналітикою для трьох ролей: студент, викладач, адміністратор.

## Запуск

1. Розпакуй архів у папку без кирилиці, наприклад `C:\projects\campuspulse-platform`
2. Відкрий цю папку у Visual Studio Code
3. У терміналі виконай:

```bash
npm install
npm start
```

4. Відкрий у браузері `http://localhost:3000`

## Демо-акаунти

- Студент: `student@campuspulse.local` / `Student123!`
- Викладач: `teacher@campuspulse.local` / `Teacher123!`
- Адмін: `admin@campuspulse.local` / `Admin123!`

## Основні можливості

- щоденник самопочуття та індекс стану
- тести з автоматичною інтерпретацією результату
- ролі студент / викладач / адміністратор
- конструктор тестів для викладача
- перегляд результатів усіх студентів для викладача
- керування користувачами та загальною статистикою для адміністратора

## Дані

Усі дані зберігаються локально у файлі `data/database.json`.
